# GitHub User Finder

**Автор:** Калугин Владимир

## Описание
GUI-приложение для поиска пользователей GitHub с использованием GitHub API. Позволяет искать пользователей, просматривать их профили и сохранять избранных пользователей в локальное хранилище.

## Функциональность
- Поиск пользователей GitHub по имени
- Отображение результатов поиска в виде таблицы
- Детальная информация о пользователе (аватар, имя, био, локация, компания)
- Добавление пользователей в избранное
- Сохранение избранных в JSON-файле
- Валидация ввода (проверка на пустые поля)

## Требования
- Python 3.6+
- Библиотеки: requests, Pillow
- Интернет-соединение
- GitHub API (бесплатный доступ без токена)
